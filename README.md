# New Age

✨ 오픈SW플랫폼 "New Age"의 “이화마켓”입니다.


**"이화마켓"** 은 이화인을 위한 중고 거래 플랫폼입니다. 

다른 중고 거래 플랫폼과는 차별화된 **'경매'** 기능을 통해 쇼핑의 즐거움을 느껴보세요 !
